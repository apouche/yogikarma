{% trans "A propos de Yogi Karma Thutop Dorje" %}

{% blocktrans %} Karma est né le 16 dec 1961 à Cuneo (Italie) sous le nom de Michel Pierini. Il est de nationalité française, et marié à Nicole Chaniac. Il vit dans le département du Var dans le village de Taradeau. {% endblocktrans %}

{% trans "Le Boudhisme" %}

{% blocktrans %} Sa rencontre avec le boudhisme tibetain date de 1999 d'où son nom tibétain qui lui a était donné par un de ses maitres et qui signifie "pouvoir indestructible". Sa Lignée est celle du Karma Kagyu dont le maître spirituel suprême est Sa Sainteté Le 17 eme Gyalwa Karmapa Ogyen Trinley Dorjé. {% endblocktrans %}

{% blocktrans %} Karma a eu la transmition complète du Mahamudra sur 7 ans, pratique de la méditation, par son maitre Le 12éme Kenting Tai Situpa Rinpoche {% endblocktrans %}

{% blocktrans %} Diplomé du Centre de Medecine tibétaine de Dharamsala (Inde, centre de SS Dalaï Lama) en Acupressure et massages. En tant que Yogi, Karma ne coupe pas ses cheveux, porte uniquement la robe de coton et pratique les Yogas Secrets de la lignée. {% endblocktrans %}

{% trans "Projets Caritatifs" %}

{% blocktrans %} Karma a réalisé un projet caritatif en Inde au profit du monastère de Tashi Jong, la dernière demeure du yogi Atrum. {% endblocktrans %}

{% blocktrans %} Le projet se portait sur la réalisation du temple qui abrite la sépulture de ce grand maître tibétain. Le montant s'élevait à 5000 euros. Yogi Karma remercie les généreux donateurs spontanés qui ont permit de finaliser ce projet. {% endblocktrans %}

{% blocktrans %} {% endblocktrans %}

{% blocktrans %} Karma mettra prochainement une boutique en ligne afin de promouvoir l'art tibétain, la totalité des bénéfices serviront à sponsoriser 2 futurs projets: {% endblocktrans %}

  • {% blocktrans %}la reconstruction du monastère de Nub Gon au Tibet pour un appel de fonds de 45000 Euros{% endblocktrans %}
  • {% blocktrans %}un projet concernant l’achat d un terrain en Inde du Sud qui servira de refuge pour les animaux{% endblocktrans %}
  • {% blocktrans %}plus amples détails seront donnés ultérieurement{% endblocktrans %}

{% trans "Enseignant des Arts Martiaux" %}

{% blocktrans %} Karma pratique les arts martiaux depuis l'age de 13 ans dont il est expert de Karaté, style Shotokan (Japon), 6éme Dan, ceinture Rouge et blanche. Il enseigne les arts martiaux depuis 30 ans et il est diplomé d'état sportif 1er degré. {% endblocktrans %}

{% blocktrans %} Son dojo actuel est celui de Sophia Antipolis dans les Alpes Maritimes. Il a formé beaucoup de ceintures noires dont certain(e)s enseignent à leur tour le karaté. {% endblocktrans %}
